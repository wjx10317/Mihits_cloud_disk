# M12 系统设置模块

## 模块概述

系统设置模块提供客户端的个性化配置能力，包括传输配置、通知管理、主题语言、自动更新、缓存清理、垃圾文件和过期文件管理。

## 数据库设计

```sql
-- 用户偏好设置
CREATE TABLE user_preferences (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL UNIQUE REFERENCES users(id) ON DELETE CASCADE,
    theme VARCHAR(20) DEFAULT 'light',        -- light/dark/system
    locale VARCHAR(10) DEFAULT 'zh-CN',
    upload_speed_limit INTEGER,                -- KB/s，NULL=不限
    download_speed_limit INTEGER,              -- KB/s，NULL=不限
    max_concurrent_uploads INTEGER DEFAULT 3,
    max_concurrent_downloads INTEGER DEFAULT 3,
    auto_sync_on_startup BOOLEAN DEFAULT FALSE,
    notification_enabled BOOLEAN DEFAULT TRUE,
    notification_sound BOOLEAN DEFAULT TRUE,
    auto_update BOOLEAN DEFAULT TRUE,
    default_upload_folder_id UUID,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- 重复文件记录
CREATE TABLE duplicate_files (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    file_hash VARCHAR(64) NOT NULL,
    file_size BIGINT NOT NULL,
    file_ids JSONB NOT NULL,                   -- 重复文件ID列表
    file_count INTEGER NOT NULL,
    wasted_space BIGINT NOT NULL,             -- 浪费的空间
    scan_id UUID NOT NULL,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- 过期文件记录
CREATE TABLE expired_files (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    file_id UUID NOT NULL REFERENCES files(id) ON DELETE CASCADE,
    file_name VARCHAR(255) NOT NULL,
    folder_path VARCHAR(1024),
    last_accessed_at TIMESTAMP WITH TIME ZONE,
    file_size BIGINT,
    days_unused INTEGER,                      -- 未访问天数
    scan_id UUID NOT NULL,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- 应用版本
CREATE TABLE app_versions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    version VARCHAR(20) NOT NULL,
    platform VARCHAR(20) NOT NULL,            -- win/mac/linux
    release_notes TEXT,
    download_url VARCHAR(512),
    file_hash VARCHAR(64),
    file_size BIGINT,
    is_mandatory BOOLEAN DEFAULT FALSE,       -- 是否强制更新
    released_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

## 子模块详细设计

### M12.1 传输配置

**前端**：
- 设置页「传输」标签页
- 配置项：
  - 上传限速：滑块 + 数字输入（单位 KB/s），0 = 不限
  - 下载限速：同上
  - 最大同时上传数：1-5
  - 最大同时下载数：1-5
  - 默认上传目录：选择网盘目录
  - 后台传输开关（最小化后继续传输）
  - 传输完成后通知
  - 传输完成后自动关机（可选）

**后端**：
- `GET /api/v1/preferences` — 获取偏好设置
- `PUT /api/v1/preferences` — 更新偏好设置
```json
{
  "upload_speed_limit": 1024,
  "download_speed_limit": null,
  "max_concurrent_uploads": 3,
  "max_concurrent_downloads": 3
}
```

**Electron 限速实现**：
- 主进程控制传输速率
- 使用令牌桶算法限速
- 每个传输任务维护独立的速率控制器

### M12.2 通知管理

**前端**：
- 设置页「通知」标签页
- 开关项：
  - 传输完成通知
  - 传输失败通知
  - 同步冲突通知
  - 分享更新通知
  - 登录安全提醒
  - 通知声音
- 使用 Electron Notification API：
```typescript
new Notification({
  title: '上传完成',
  body: 'video.mp4 已成功上传',
  icon: path.join(__dirname, 'icon.png'),
  silent: !preferences.notificationSound
}).show()
```

### M12.3 主题语言

**前端**：
- 设置页「外观」标签页
- 主题切换：
  - 浅色 / 深色 / 跟随系统
  - 切换时平滑过渡动画
- 语言切换：
  - 简体中文 / English
  - 使用 vue-i18n 实现国际化
- Element Plus 主题色定制

**实现**：
```typescript
// 主题切换
const applyTheme = (theme: 'light' | 'dark' | 'system') => {
  if (theme === 'system') {
    const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches
    document.documentElement.setAttribute('data-theme', prefersDark ? 'dark' : 'light')
  } else {
    document.documentElement.setAttribute('data-theme', theme)
  }
}
```

### M12.4 自动更新

**前端**：
- 设置页「关于」→ 自动更新开关
- 更新流程：
  1. 应用启动时检查更新
  2. 发现新版本 → 弹窗展示更新日志
  3. 用户选择「立即更新」/「稍后提醒」
  4. 下载更新包 → 安装重启
- 强制更新：is_mandatory = true 时必须更新才能使用

**后端**：
- `GET /api/v1/app/updates?platform=win&current_version=1.0.0` — 检查更新
- 响应：
```json
{
  "code": "SUCCESS",
  "data": {
    "has_update": true,
    "latest_version": "1.1.0",
    "release_notes": "1. 新增文件同步功能\n2. 修复上传进度条异常",
    "download_url": "https://...",
    "file_hash": "sha256...",
    "file_size": 83886080,
    "is_mandatory": false
  }
}
```

**Electron 实现**：
- 使用 electron-updater（electron-builder 自带）
- 支持增量更新（差量下载）
- 下载进度展示
- 安装包签名校验

### M12.5 缓存清理

**前端**：
- 设置页「存储」→ 缓存管理
- 展示项：
  - 缩略图缓存大小
  - 离线文件缓存大小
  - 临时文件大小
  - 日志文件大小
  - 总缓存大小
- 操作：一键清理 / 按类别清理
- 清理前确认弹窗

**后端**：
- `GET /api/v1/cache/stats` — 缓存统计
- `DELETE /api/v1/cache/thumbnails` — 清理缩略图缓存
- `DELETE /api/v1/cache/temp` — 清理临时文件

**Electron 实现**：
- 扫描缓存目录大小
- 按类别统计
- 删除后刷新统计

### M12.6 垃圾文件

**前端**：
- 设置页「存储」→ 垃圾文件
- 「扫描」按钮 → 异步扫描
- 扫描结果分类展示：
  - 重复文件组：每组展示文件列表、占用空间
  - 建议：保留最新的，删除其他重复
- 勾选要清理的文件 → 确认删除

**后端**：
- `POST /api/v1/storage/scan/duplicates` — 触发重复文件扫描（异步）
- `GET /api/v1/storage/scan/{scan_id}/status` — 扫描进度
- `GET /api/v1/storage/scan/{scan_id}/results` — 扫描结果
- `POST /api/v1/storage/clean/duplicates` — 清理重复文件
```json
{ "keep_strategy": "newest", "scan_id": "uuid" }
```

**重复检测算法**：
1. 按文件大小分组（size 相同才可能重复）
2. 同组内比对文件哈希（SHA-256）
3. 哈希相同 → 确认为重复文件

### M12.7 过期文件

**前端**：
- 设置页「存储」→ 过期文件
- 可配置过期天数：30 / 60 / 90 / 180 / 365 天
- 「扫描」按钮 → 扫描长期未访问的文件
- 结果列表：文件名、路径、大小、最后访问时间
- 操作：删除 / 归档

**后端**：
- `POST /api/v1/storage/scan/expired` — 扫描过期文件
```json
{ "days_unused": 90 }
```
- `GET /api/v1/storage/scan/{scan_id}/results` — 扫描结果
- `POST /api/v1/storage/clean/expired` — 清理过期文件

## 错误码

| 错误码 | HTTP状态 | 说明 |
|--------|----------|------|
| SCAN_IN_PROGRESS | 409 | 扫描正在进行中 |
| SCAN_NOT_FOUND | 404 | 扫描任务不存在 |
| UPDATE_NOT_AVAILABLE | 200 | 无可用更新 |
| CACHE_CLEAN_FAILED | 500 | 缓存清理失败 |
