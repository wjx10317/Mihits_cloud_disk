# M13 辅助集成模块

## 模块概述

辅助集成模块提供网盘客户端与操作系统的深度集成能力，包括系统托盘、开机自启、系统右键菜单、文件关联、会员套餐管理。增强客户端的系统级体验。

## 子模块详细设计

### M13.1 系统托盘

**前端**：
- 托盘图标：网盘 Logo
- 最小化到托盘：关闭窗口时最小化而非退出
- 托盘右键菜单：
  - 打开主窗口
  - 传输状态（N 个上传 / M 个下载）
  - 暂停所有传输
  - 同步状态
  - 设置
  - 退出

**Electron 实现**：
```typescript
import { Tray, Menu, nativeImage } from 'electron'

class AppTray {
  private tray: Tray

  constructor() {
    const icon = nativeImage.createFromPath(path.join(__dirname, 'tray-icon.png'))
    this.tray = new Tray(icon)
    this.tray.setToolTip('Mihits 网盘')
    this.updateMenu()
  }

  updateMenu(transfers?: TransferStatus) {
    const contextMenu = Menu.buildFromTemplate([
      { label: '打开网盘', click: () => this.showWindow() },
      { type: 'separator' },
      { label: `上传: ${transfers?.uploading ?? 0} 个`, enabled: false },
      { label: `下载: ${transfers?.downloading ?? 0} 个`, enabled: false },
      { label: '暂停所有传输', click: () => this.pauseAll() },
      { type: 'separator' },
      { label: '设置', click: () => this.showSettings() },
      { type: 'separator' },
      { label: '退出', click: () => app.quit() }
    ])
    this.tray.setContextMenu(contextMenu)
  }
}
```

- 托盘图标动态：传输中显示动态图标或覆盖数字角标

### M13.2 开机自启

**前端**：
- 设置页「通用」→ 开机自启动开关
- 登录后自动启动（可选）

**Electron 实现**：
```typescript
import app from 'electron'
import { app as electronApp } from 'electron'

// 设置开机自启
function setAutoLaunch(enable: boolean) {
  app.setLoginItemSettings({
    openAtLogin: enable,
    path: electronApp.getPath('exe'),
    args: ['--hidden']  // 启动时最小化到托盘
  })
}
```

- `--hidden` 参数：启动时最小化到托盘，不显示主窗口

### M13.3 系统右键菜单

**前端**：
- 设置页「集成」→ 系统右键菜单开关
- 注册右键菜单项：
  - 「上传到 Mihits 网盘」→ 选择文件后上传到默认目录

**Windows 实现**：
```typescript
// 注册注册表项
import Registry from 'winreg'

function registerContextMenu() {
  const regKey = new Registry({
    hive: Registry.HKCU,
    key: '\\Software\\Classes\\*\\shell\\MihitsUpload'
  })
  
  regKey.set('', Registry.REG_SZ, '上传到 Mihits 网盘', (err) => {
    regKey.set('icon', Registry.REG_SZ, appPath, () => {})
    regKey.set('command', '', Registry.REG_SZ, `"appPath}" --upload "%1"`, () => {})
  })
}
```

**macOS 实现**：
- 使用 Finder Sync Extension（需额外开发 Extension）
- 或使用 Automator 快速操作作为替代方案

### M13.4 文件关联

**前端**：
- 设置页「集成」→ 文件关联
- 可关联的文件类型列表（勾选）：
  - 网盘支持的预览格式：.txt, .md, .pdf, .jpg, .png, .mp4 等
  - 双击关联文件 → 用网盘客户端打开预览

**实现**：
- Windows：注册表关联文件类型
- macOS：Info.plist 中声明 document types
- 接收文件打开事件：
```typescript
app.on('open-file', (event, filePath) => {
  event.preventDefault()
  // 在网盘中查找并预览该文件
  mainWindow.webContents.send('file:open', filePath)
})
```

### M13.5 会员套餐

**前端**：
- 侧边栏「会员」入口
- 套餐展示页：
  - 免费版：5GB 存储
  - 基础版：100GB / ¥9.9/月
  - 专业版：1TB / ¥29.9/月
  - 旗舰版：5TB / ¥59.9/月
- 容量扩容按钮 → 支付流程
- 当前套餐信息、到期时间
- 使用量对比（当前用量 vs 套餐容量）

**后端**：
- `GET /api/v1/plans` — 套餐列表
- `GET /api/v1/users/me/subscription` — 当前订阅
- `POST /api/v1/subscriptions/upgrade` — 升级套餐
```json
{ "plan_id": "uuid", "payment_method": "alipay|wechat" }
```
- `POST /api/v1/subscriptions/cancel` — 取消订阅
- `GET /api/v1/subscriptions/history` — 订阅历史

**数据库设计**：
```sql
CREATE TABLE plans (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name VARCHAR(50) NOT NULL,
    storage_quota BIGINT NOT NULL,
    price_monthly DECIMAL(10, 2),            -- 月价格
    price_yearly DECIMAL(10, 2),             -- 年价格
    features JSONB,                          -- 功能列表
    is_active BOOLEAN DEFAULT TRUE,
    sort_order INTEGER DEFAULT 0,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

CREATE TABLE subscriptions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    plan_id UUID NOT NULL REFERENCES plans(id),
    status VARCHAR(20) DEFAULT 'active',     -- active/expired/cancelled
    started_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    expires_at TIMESTAMP WITH TIME ZONE,
    auto_renew BOOLEAN DEFAULT TRUE,
    payment_method VARCHAR(20),
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

CREATE TABLE payment_records (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    subscription_id UUID REFERENCES subscriptions(id),
    amount DECIMAL(10, 2) NOT NULL,
    payment_method VARCHAR(20) NOT NULL,
    payment_status VARCHAR(20) DEFAULT 'pending', -- pending/success/failed/refunded
    transaction_id VARCHAR(255),
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

## API 汇总

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/v1/plans | 套餐列表 |
| GET | /api/v1/users/me/subscription | 当前订阅 |
| POST | /api/v1/subscriptions/upgrade | 升级套餐 |
| POST | /api/v1/subscriptions/cancel | 取消订阅 |
| GET | /api/v1/subscriptions/history | 订阅历史 |

## 错误码

| 错误码 | HTTP状态 | 说明 |
|--------|----------|------|
| PLAN_NOT_FOUND | 404 | 套餐不存在 |
| SUBSCRIPTION_ACTIVE | 409 | 已有有效订阅 |
| PAYMENT_FAILED | 402 | 支付失败 |
| DOWNGRADE_NOT_ALLOWED | 400 | 当前用量超出目标套餐容量 |
