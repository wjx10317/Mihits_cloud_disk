# M5 基础架构模块

## 模块概述

基础架构模块是整个网盘客户端的骨架，包括 Electron 主进程、路由框架、状态管理、网络通信层和本地持久化。所有业务模块都依赖此模块。

## 技术栈确认

| 层面 | 技术 | 版本 |
|------|------|------|
| 桌面框架 | Electron | ≥ 28.x |
| 构建工具 | electron-vite | ≥ 2.x |
| 前端框架 | Vue 3 | ≥ 3.4 |
| UI 组件库 | Element Plus | ≥ 2.5 |

### M5.1 Electron 主进程

**项目结构**：

**窗口管理**：
- 主窗口：1200x800 最小尺寸 900x600
- 启动时居中显示
- 关闭时最小化到托盘（非退出）
- 单实例运行（通过 app.requestSingleInstanceLock）

### M5.4 网络层

**Axios 实例配置**：
```typescript
const apiClient = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL,
  timeout: 30000,
  headers: { 'Content-Type': 'application/json' }
})

// 请求拦截器
apiClient.interceptors.request.use((config) => {
  const userStore = useUserStore()
  if (userStore.accessToken) {
    config.headers.Authorization = `Bearer ${userStore.accessToken}`
  }
  return config
})

// 响应拦截器
apiClient.interceptors.response.use(
  (response) => response.data,
  async (error) => {
    if (error.response?.status === 401) {
      const code = error.response.data?.code
      if (code === 'TOKEN_EXPIRED') {
        // 尝试刷新 Token
        try {
          await useUserStore().refreshAccessToken()
          return apiClient(error.config)  // 重试原请求
        } catch {
          useUserStore().logout()
          router.push('/login')
        }
      }
    }
    return Promise.reject(error)
  }
)
```

**错误统一处理**：
- 所有后端错误返回 `{ code: string, message: "中文提示" }`
- 前端根据 code 分类处理，message 直接展示给用户
- 网络错误统一弹窗提示"网络连接异常，请检查网络设置"
