# M8 文件预览模块

## 模块概述

文件预览模块支持多种文件格式的在线预览，包括图片、文档、音视频、纯文本和压缩文件，并提供基础的在线编辑能力。是提升用户体验的核心功能。

## 技术选型

| 文件类型 | 预览技术 | 编辑技术 |
|----------|----------|----------|
| 图片 | 内置 Image Viewer + 缩放/轮播 | Canvas 简易标注/裁剪 |
| PDF | pdf.js | — |
| Office 文档 | LibreOffice Headless 转码 → PDF → pdf.js | — |
| 纯文本 | Monaco Editor（只读模式） | Monaco Editor（编辑模式） |
| 音视频 | HTML5 Video/Audio + 自定义控件 | — |
| 压缩文件 | 后端解压列出文件树 | — |

## 子模块详细设计

### M8.1 图片预览

**前端**：
- 点击图片文件 → 打开全屏预览
- 功能：
  - 缩放（鼠标滚轮 / 按钮放大缩小）
  - 旋转（90° 旋转）
  - 左右轮播（同目录下所有图片）
  - 原图 / 压缩预览切换
  - 下载原图按钮

**后端**：
- `GET /api/v1/files/{id}/preview` — 获取预览链接
- 响应：
```json
{
  "code": "SUCCESS",
  "data": {
    "preview_url": "签名URL",
    "thumbnail_url": "缩略图签名URL",
    "original_url": "原图签名URL",
    "width": 1920,
    "height": 1080,
    "format": "jpg"
  }
}
```

### M8.2 文档预览

**前端**：
- PDF 文件：使用 pdf.js 渲染
- Word/Excel/PPT：提示转码中，后端异步转码为 PDF 后展示
- 页码导航、搜索、缩放

**后端**：
- `GET /api/v1/files/{id}/preview/document`
- 逻辑：
  1. 检查是否已有转码缓存（PDF 版本）
  2. 未转码 → 加入转码队列，返回 `CONVERTING` 状态
  3. 转码中使用 LibreOffice Headless：
```bash
libreoffice --headless --convert-to pdf input.docx --outdir /tmp/
```
  4. 转码完成后上传 PDF 到 MinIO（缓存）
  5. 返回 PDF 签名 URL

- 转码状态查询：`GET /api/v1/files/{id}/preview/status`

**转码缓存表**：
```sql
CREATE TABLE document_conversions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    file_id UUID NOT NULL REFERENCES files(id) ON DELETE CASCADE,
    pdf_storage_key VARCHAR(512),
    status VARCHAR(20) DEFAULT 'pending',  -- pending/converting/completed/failed
    error_message TEXT,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    CONSTRAINT unique_file_conversion UNIQUE (file_id)
);
```

### M8.3 音视频

**前端**：
- 自定义播放器（基于 HTML5 Video/Audio）
- 功能：
  - 播放/暂停
  - 进度条拖拽
  - 倍速播放（0.5x / 0.75x / 1x / 1.25x / 1.5x / 2x）
  - 音量控制
  - 全屏切换
  - 画中画模式

**后端**：
- `GET /api/v1/files/{id}/preview/media`
- 返回流式签名 URL（支持 Range 请求）
- 对于大视频，后端可选转码为 MP4（H.264）以兼容浏览器播放

### M8.4 文本预览

**前端**：
- 使用 Monaco Editor（只读模式）展示
- 语法高亮（自动检测语言类型）
- 行号显示
- 字符编码自动检测

**后端**：
- `GET /api/v1/files/{id}/preview/text`
- 返回文本内容（限制前 1MB，超出部分截断提示）
- 响应：
```json
{
  "code": "SUCCESS",
  "data": {
    "content": "文件内容...",
    "language": "python",
    "encoding": "utf-8",
    "size": 2048,
    "truncated": false
  }
}
```

### M8.5 缩略图

**前端**：
- 网格视图展示文件缩略图
- 懒加载：滚动到可视区域再请求
- 缩略图占位符：按文件类型显示默认图标

**后端**：
- `GET /api/v1/files/{id}/thumbnail?size=128` — 获取缩略图
- 支持尺寸：64 / 128 / 256 / 512
- 生成逻辑：
  1. 首次请求时生成缩略图（Pillow / Sharp）
  2. 缓存到 MinIO（`thumbnails/{file_id}_{size}.webp`）
  3. 后续请求直接返回缓存

**缩略图缓存表**：
```sql
CREATE TABLE thumbnail_cache (
    file_id UUID PRIMARY KEY REFERENCES files(id) ON DELETE CASCADE,
    storage_key VARCHAR(512) NOT NULL,
    size INTEGER NOT NULL,
    generated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

### M8.6 压缩文件

**前端**：
- 压缩文件预览面板
- 树形展示压缩包内文件列表
- 双击文件 → 预览 / 单独下载
- 「全部解压」按钮：选择目标目录后解压到网盘

**后端**：
- `GET /api/v1/files/{id}/preview/archive` — 列出压缩包内容
- 响应：
```json
{
  "code": "SUCCESS",
  "data": {
    "format": "zip",
    "file_count": 15,
    "total_size": 5242880,
    "entries": [
      { "path": "docs/readme.txt", "size": 1024, "is_dir": false },
      { "path": "docs/images/", "size": 0, "is_dir": true }
    ]
  }
}
```
- `POST /api/v1/files/{id}/extract` — 解压到指定目录
```json
{ "target_folder_id": "uuid", "entries": ["docs/readme.txt"] }
```

### M8.7 在线编辑

**前端**：
- 文本文件：Monaco Editor 编辑模式
  - 自动保存（debounce 1s）
  - Ctrl+S 手动保存
  - 未保存提示
- Markdown 文件：分栏预览（左编辑右预览）

**后端**：
- `PUT /api/v1/files/{id}/content` — 保存文件内容
```json
{ "content": "更新后的内容..." }
```
- 逻辑：
  1. 验证编辑权限
  2. 创建文件新版本（MinIO 版本控制）
  3. 更新 file 记录
  4. 更新 storage_used（如大小变化）

## API 汇总

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/v1/files/{id}/preview | 获取预览信息 |
| GET | /api/v1/files/{id}/preview/document | 文档预览 |
| GET | /api/v1/files/{id}/preview/status | 转码状态 |
| GET | /api/v1/files/{id}/preview/media | 音视频预览 |
| GET | /api/v1/files/{id}/preview/text | 文本预览 |
| GET | /api/v1/files/{id}/thumbnail | 获取缩略图 |
| GET | /api/v1/files/{id}/preview/archive | 压缩文件内容 |
| POST | /api/v1/files/{id}/extract | 解压文件 |
| PUT | /api/v1/files/{id}/content | 保存文件内容 |

## 错误码

| 错误码 | HTTP状态 | 说明 |
|--------|----------|------|
| PREVIEW_NOT_SUPPORTED | 400 | 该文件类型不支持预览 |
| CONVERTING | 202 | 文件正在转码中 |
| CONVERSION_FAILED | 500 | 文件转码失败 |
| FILE_TOO_LARGE | 400 | 文件过大，无法预览 |
| ARCHIVE_PASSWORD_REQUIRED | 403 | 压缩文件需要密码 |

## 测试要求

- 覆盖率目标：≥ 94%（复杂任务）
- 关键测试场景：
  - 图片预览和缩放
  - PDF 文件渲染
  - Office 文档转码流程
  - 音视频播放和倍速
  - 文本预览语法高亮
  - 缩略图生成和缓存
  - 压缩文件列表和解压
  - 在线编辑保存
