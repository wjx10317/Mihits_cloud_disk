# M7 分享协作模块

## 模块概述

分享协作模块实现文件和文件夹的分享功能，支持链接分享、权限控制、分享管理和协作文件夹。是网盘从个人工具升级为协作平台的关键模块。

## 数据库设计

```sql
-- 分享记录
CREATE TABLE shares (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    file_id UUID REFERENCES files(id) ON DELETE CASCADE,
    folder_id UUID REFERENCES folders(id) ON DELETE CASCADE,
    share_code VARCHAR(8) UNIQUE NOT NULL,   -- 短分享码
    extract_code VARCHAR(6),                 -- 提取码（可选）
    permission VARCHAR(20) NOT NULL DEFAULT 'view',  -- view/edit/download
    expires_at TIMESTAMP WITH TIME ZONE,     -- 过期时间（NULL=永不过期）
    access_count INTEGER DEFAULT 0,
    max_access_count INTEGER,                -- 最大访问次数（NULL=无限制）
    is_active BOOLEAN DEFAULT TRUE,
    password_protected BOOLEAN DEFAULT FALSE,
    allow_download BOOLEAN DEFAULT TRUE,
    watermark BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- 分享访问记录
CREATE TABLE share_access_logs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    share_id UUID NOT NULL REFERENCES shares(id) ON DELETE CASCADE,
    visitor_ip VARCHAR(45),
    visitor_name VARCHAR(100),               -- 访客名称（非注册用户）
    access_type VARCHAR(20),                 -- view/download/edit
    accessed_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- 协作文件夹
CREATE TABLE collaboration_folders (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    folder_id UUID NOT NULL REFERENCES folders(id) ON DELETE CASCADE,
    owner_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    name VARCHAR(255) NOT NULL,
    description TEXT,
    is_active BOOLEAN DEFAULT TRUE,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- 协作成员
CREATE TABLE collaboration_members (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    collaboration_id UUID NOT NULL REFERENCES collaboration_folders(id) ON DELETE CASCADE,
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    role VARCHAR(20) NOT NULL DEFAULT 'viewer',  -- owner/admin/editor/viewer
    joined_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
    CONSTRAINT unique_collab_member UNIQUE (collaboration_id, user_id)
);

-- 协作邀请
CREATE TABLE collaboration_invitations (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    collaboration_id UUID NOT NULL REFERENCES collaboration_folders(id) ON DELETE CASCADE,
    inviter_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    invitee_email VARCHAR(255) NOT NULL,
    role VARCHAR(20) NOT NULL DEFAULT 'viewer',
    token VARCHAR(255) UNIQUE NOT NULL,      -- 邀请令牌
    status VARCHAR(20) DEFAULT 'pending',    -- pending/accepted/rejected/expired
    expires_at TIMESTAMP WITH TIME ZONE,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);
```

## 子模块详细设计

### M7.1 链接分享

**前端**：
- 右键菜单「分享」→ 分享弹窗
- 配置项：
  - 提取码：自动生成 6 位 / 自定义 / 无提取码
  - 有效期：1天 / 7天 / 30天 / 永久
  - 访问权限：仅查看 / 可编辑 / 可下载
- 生成分享链接：`https://cloud.mihits.com/s/{share_code}`
- 一键复制链接+提取码

**后端**：
- `POST /api/v1/shares` — 创建分享
```json
{
  "file_id": "uuid",
  "folder_id": "uuid",
  "has_extract_code": true,
  "expires_in": 86400,
  "permission": "view",
  "allow_download": true
}
```
- 响应：
```json
{
  "code": "SUCCESS",
  "data": {
    "share_code": "Ab3Kx9",
    "share_url": "https://cloud.mihits.com/s/Ab3Kx9",
    "extract_code": "5m8P2q"
  }
}
```
- `GET /api/v1/shares/{share_code}` — 获取分享信息（需提取码验证）
- `POST /api/v1/shares/{share_code}/verify` — 验证提取码

### M7.2 权限控制

**权限层级**：

| 权限 | 查看 | 下载 | 编辑 | 删除 | 分享 |
|------|------|------|------|------|------|
| view | ✅ | ❌ | ❌ | ❌ | ❌ |
| edit | ✅ | ✅ | ✅ | ❌ | ❌ |
| admin | ✅ | ✅ | ✅ | ✅ | ✅ |

**实现**：
- 后端中间件校验分享权限
- 编辑权限：可上传新文件、修改文件内容
- 下载权限：可获取下载签名 URL
- 权限与分享记录绑定，修改分享即修改权限

### M7.3 分享管理

**前端**：
- 侧边栏「我的分享」页面
- 列表展示：文件名、权限、访问次数、创建时间、过期时间
- 操作：复制链接、编辑设置、取消分享
- 筛选：进行中 / 已过期

**后端**：
- `GET /api/v1/shares` — 我的分享列表
- `PUT /api/v1/shares/{id}` — 更新分享设置
- `DELETE /api/v1/shares/{id}` — 取消分享
- `GET /api/v1/shares/{id}/access-logs` — 访问记录

### M7.4 协作文件夹

**前端**：
- 文件夹右键菜单「创建协作文件夹」
- 协作管理面板：
  - 成员列表 + 角色标签
  - 邀请新成员（输入邮箱）
  - 修改成员权限
  - 移除成员
- 协作文件夹特殊图标标识

**后端**：
- `POST /api/v1/collaborations` — 创建协作文件夹
- `GET /api/v1/collaborations` — 我的协作列表
- `POST /api/v1/collaborations/{id}/invite` — 邀请成员
```json
{ "email": "user@example.com", "role": "editor" }
```
- `PUT /api/v1/collaborations/{id}/members/{userId}` — 修改权限
- `DELETE /api/v1/collaborations/{id}/members/{userId}` — 移除成员
- `POST /api/v1/collaborations/{id}/accept` — 接受邀请
- `DELETE /api/v1/collaborations/{id}` — 解散协作

### M7.5 防泄露

**前端**：
- 分享设置中可开启：
  - 禁止下载
  - 水印叠加（查看时显示访问者信息水印）
  - 访问次数限制

**后端**：
- 禁止下载：不返回签名 URL，仅支持在线预览
- 水印：返回文件时叠加半透明水印（图片型）或 CSS 水印层（文档型）
- 访问次数：每次访问 +1，达到上限后 is_active = FALSE

## 错误码

| 错误码 | HTTP状态 | 说明 |
|--------|----------|------|
| SHARE_NOT_FOUND | 404 | 分享不存在 |
| SHARE_EXPIRED | 410 | 分享已过期 |
| EXTRACT_CODE_INVALID | 403 | 提取码错误 |
| SHARE_ACCESS_LIMIT | 403 | 分享访问次数已达上限 |
| PERMISSION_DENIED | 403 | 权限不足 |
| ALREADY_MEMBER | 409 | 已是协作成员 |
| INVITATION_EXPIRED | 410 | 邀请已过期 |
